BACKUP VIEWER

VIEW YOUR BACKUP'S MESSAGE AND CALL LOG YBK FILES ON COMPUTER.

Been using YUH APPS' BACKUP for a while and now you need to view the message and call log backup files on computer? BACKUP VIEWER is now ready to do the job. Just simply drag and drop a YBK file to either the message or call icon and drop it there to reveal its content in a few seconds. Of course, the YBK files have to be on your computer.

The app works on all computer platforms: Windows, Mac and Linux. So whatever the OS your computer is on, BACKUP VIEWER is there to do what it is made for.

(*) Only message and call log backups are viewable by the app. Wifi passwords and Bluetooth pairs are not supported.

(*) Backup files don't contain your contacts' names. It only contain their phone number. Therefore, no contact names will be shown.
