BASE64 Encoder & Decoder
What is this tool?

As the name already suggests, Yuh Apps' Base64 Encoder & Decoder helps you easily encode a String, with an optional key to XOR the characters, and of course, the opposite which decodes a Base64 String with the provided key.

For example, encode 'Hello World' will give you 'SGVsbG8gV29ybGQ=' without the key, or 'KQkcBA5BOx8aDQU=' with the key as 'alpha'.

This tool is compatible with other languages like Java and Swift. That means if you encode a String here then decode using a Java or Swift program, you'll get the exact same result.
